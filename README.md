# 🎓 CollegeBuzz

CollegeBuzz is a comprehensive college community platform designed to make campus life easier, smarter, and more connected. It provides students with a single place to access various campus services such as Events, Clubs, Lost & Found, Second-Hand Marketplace, Cab Sharing, and Complaint Management.

The platform focuses on improving communication between students while providing an easy-to-use interface and secure authentication system.

# 🚀 Installation

Clone the repository

```bash
git clone <repository-url>
```

Install dependencies

```bash
npm install
```

Start the server

```bash
node app.js
```

or

```bash
npm start
```
